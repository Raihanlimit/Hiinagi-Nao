# 陽凪なお — Hiinagi Nao

Nao is a persistent WhatsApp AI assistant built around natural interaction, memory, autonomous behavior, and human-like presence.

Unlike traditional chatbots, Nao is designed to feel alive:
- natural typing & delay
- mood-based responses
- group memory
- voice interaction
- autonomous maintenance
- multilingual personality

---

## Features

- WhatsApp integration
- DeepSeek conversation engine
- Gemini & Google Vision media analysis
- Tavily web search
- Voice message support
- Human-like typing presence
- Group & private memory
- Autonomous self-maintenance
- Media downloader & scraper
- Dynamic mood system

---

## Modes

| Mode | Description |
|------|-------------|
| normal | default behavior |
| developer | technical/debug mode |
| custom | active only in selected chats |
| self | owner-only interaction |
| public | active everywhere |
